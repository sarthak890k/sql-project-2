# ☕ Coffee Shop Sales Analysis using SQL

## 📌 Project Overview
This project analyzes coffee shop sales data using SQL to uncover business insights related to revenue, product performance, customer purchasing behavior, and store operations.

The objective is to demonstrate SQL skills such as data exploration, aggregation, grouping, filtering, joins , date functions, and business analysis.

---

## 🛠️ Tools Used

- MySQL
- SQL
- MySQL Workbench

---

## 📂 Dataset

The dataset contains coffee shop transaction records including:

- Transaction ID
- Transaction Date
- Product Category
- Product Type
- Store Location
- Quantity Sold
- Unit Price
- Total Sales

---

## 🎯 Business Objectives

- Analyze overall sales performance
- Identify top-selling products
- Compare store performance
- Understand customer purchasing trends
- Generate actionable business insights

## 📌 Key Insights

- Coffee generated the highest overall revenue.
- Morning hours recorded the maximum number of orders.
- Weekend sales were higher than weekdays.
- Certain store locations consistently outperformed others.
- A small number of products contributed a large share of total sales.
